Confer

Confer is an online dynamic discussion board , brings in live experience of group discussion and debate.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Confer.

I have this idea of coming up with a dynamic website ,holding all powers of a group discussion and debate,at the same time providing live experience and instant connectivity with other people taking part in these activites:


#to clone this project
$ git clone git://github.com/tuhinrawat/confer.git

#to enter in the working directory
$ cd confer
